## Recreate 1.0.0

The first public release of Recreate — a desktop app that turns a topic into a finished video, end to end.

### What it does
Give Recreate an idea and it handles the whole pipeline:

- ✍️ **Script** — writes the script with an LLM (DeepSeek)
- 🎙️ **Voiceover** — generates narration via MiniMax or Edge TTS
- 🎬 **Visuals** — pulls matching stock footage and images from Pexels and Pixabay
- 🔊 **Audio** — sources background/sound effects from Freesound
- 🪄 **Compose** — stitches it all into a final video with smooth image animations

### Highlights
- Organize your work into folders and sessions in the sidebar
- Live download progress while media is fetched
- Configurable output directory and per-provider API keys in Settings
- Built-in auto-updates — Recreate checks for new versions on launch and updates with one click

### Install
Download the installer for your architecture from the [releases page](https://github.com/ammarrik/recreate-app/releases) and run it:

- **Recreate-Setup-1.0.0-x64.exe** — Intel/AMD (64-bit)
- **Recreate-Setup-1.0.0-arm64.exe** — ARM (e.g. Snapdragon)

Windows may show a SmartScreen warning since the installer isn't code-signed yet — choose **More info → Run anyway**.
